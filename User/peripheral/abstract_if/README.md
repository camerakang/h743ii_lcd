# 抽象接口

## 1 简介

抽象接口文件夹中存放的是抽象接口文件，这些文件定义了各个模块的抽象接口，用于实现模块之间的解耦。

## 2 文件列表

| 文件名 | 接口类名称 | 说明 |
| :----: | :----: | ------ |
| normalized_sensor_impl.h | NormalizedSensorImpl | 用于支持如推杆、摇杆、手轮等类型的传感器，其实现基于此接口 |
